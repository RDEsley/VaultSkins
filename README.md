# 📌 Sobre o Projeto

O **VaultSkins** é uma aplicação web desenvolvida para **gerenciar skins de jogos** através de um painel administrativo.

A aplicação permite **adicionar, visualizar, editar e remover skins**, funcionando como um **sistema CRUD completo**, com interface moderna e integração com banco de dados.

> 🎯 O objetivo do projeto é praticar conceitos de **desenvolvimento Full Stack**, incluindo API, banco de dados e interface web.

---

# ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 🧾 **Listagem de Skins** | Visualização de todas as skins cadastradas |
| ➕ **Adicionar Skin** | Cadastro de novas skins no sistema |
| ✏️ **Editar Skin** | Atualização de informações existentes |
| ❌ **Excluir Skin** | Remoção de skins do banco de dados |
| 🔐 **Painel Administrativo** | Controle completo para gerenciamento |
| 💾 **Persistência de Dados** | Armazenamento usando banco de dados |

# 🚀 Como Executar

## Pré-requisitos

Antes de começar, você precisará ter instalado:

- Node.js (versão recente)
- MongoDB
- Git

---

## Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/RDEsley/VaultSkins.git

# 2. Entre na pasta do projeto
cd VaultSkins

# 3. Instale as dependências
npm install

# 4. Inicie o servidor
npm start
```

A aplicação estará disponível em:

http://localhost:3000
